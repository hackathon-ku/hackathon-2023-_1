คำสั่ง git พื้นฐานในการอัปโหลดโค้ดขึ้นมาที่ repository นี้


```bash
git add .
git commit -m "<commit_message>"
git push -u origin main
```
คำสั่ง run 
```bash
npm install
cd server
pip install -r requirement.txt
```
